# Release v0.1.105 / 发布说明

## English

### Summary

- v0.1.105 is a reliability patch for restart-safe Matrix session recovery and release verification.

### Highlights

- Validate persisted Matrix session workdirs before reuse and fall back safely when the old path no longer exists or is invalid.
- Expose the effective workdir and its source in `/status`, and include persisted Codex workdirs in admin session history exports.
- Auto-fallback Playwright e2e to bundled Chromium when system Chrome is unavailable, and add docs consistency checks to `release:verify`.

### Suggested verification (optional)

1. Upgrade: `npm install -g codeharbor@0.1.105`
2. Check version: `codeharbor --version`
3. Restart the service, then send `/status` in a previously active Matrix room/session.
4. Confirm the reported `workdir` and `workdirSource` reflect the resumed session or a clear fallback to the room default.

## 中文

### 摘要

- v0.1.105 是一个聚焦于 Matrix 会话重启恢复和发布校验可靠性的补丁版本。

### 重点更新

- 恢复 Matrix 会话前会先校验持久化的 workdir；如果原路径不存在或已无效，会安全回退到房间默认目录。
- `/status` 现在会显示当前实际 workdir 及其来源，管理端会话历史导出也会带出持久化的 Codex workdir。
- Playwright e2e 在系统 Chrome 不存在时会自动回退到 bundled Chromium，并为 `release:verify` 增加文档一致性校验。

### 推荐验证（可选）

1. 升级：`npm install -g codeharbor@0.1.105`
2. 检查版本：`codeharbor --version`
3. 重启服务后，在之前活跃过的 Matrix room/session 中发送 `/status`
4. 确认返回中的 `workdir` 和 `workdirSource` 能正确反映会话恢复结果，或明确显示已回退到房间默认目录。
