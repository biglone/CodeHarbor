# GitHub Social Preview 上传指南（中文）

已生成可直接上传的封面图：

- `docs/assets/social-preview-1200x630.png`
- 尺寸：`1200x630`
- 大小：约 `56KB`

## 上传步骤

1. 打开仓库：`https://github.com/biglone/CodeHarbor`
2. 进入：`Settings` -> `General` -> `Social preview`
3. 点击 `Upload an image`，选择：
   - `docs/assets/social-preview-1200x630.png`
4. 保存后，刷新仓库首页并检查链接卡片预览。

## 建议复核

- 在 GitHub、微信、X/微博、Slack 中粘贴仓库链接，看卡片是否清晰。
- 如果后续定位变化（例如支持新后端/新功能），建议同步更新标题与标签后再上传新图。
