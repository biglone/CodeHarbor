# Release v0.1.105

## Highlights

- Hardened Matrix session workdir recovery: persisted session workdirs are now validated before reuse, with automatic fallback to the room default when the stored path is no longer usable.
- Improved runtime visibility: `/status` now shows the effective workdir and whether it comes from the resumed session or the room default, and admin session history exports include the persisted Codex workdir.
- Improved release verification reliability: Playwright e2e now auto-falls back to bundled Chromium when system Chrome is unavailable, and `release:verify` adds docs consistency checks to mirror CI.

## Upgrade

```bash
npm install -g codeharbor@0.1.105
```

## Verify

```bash
codeharbor --version
```

Expected output:

```text
0.1.105
```

## Notes

- This patch is recommended if you rely on long-lived Matrix room sessions and want restart-safe workspace continuity with clearer diagnostics.
- After upgrade, you can verify recovery behavior by restarting the service, sending `/status`, and checking that the reported `workdir` and `workdirSource` match the resumed session state.
