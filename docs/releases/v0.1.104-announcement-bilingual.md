# Release v0.1.104 / 发布说明

## English

### Summary

- v0.1.104 is a focused reliability patch for Matrix session recovery after service restart.

### Highlights

- Fixed resumed Matrix sessions falling back to the room default workdir after process restart.
- Persisted the effective chat workdir together with session state so the same room/user conversation continues in the original workspace.
- Added SQLite migration support and restart regression coverage for the recovery path.

### Suggested verification (optional)

1. Upgrade: `npm install -g codeharbor@0.1.104`
2. Check version: `codeharbor --version`
3. In a Matrix room/session that already used a custom workdir, restart the service and send another message.
4. Confirm the bot still runs in the original workspace instead of the default room workdir.

## 中文

### 摘要

- v0.1.104 是一个面向 Matrix 会话恢复可靠性的补丁版本，重点修复服务重启后的工作目录回退问题。

### 重点更新

- 修复服务重启后恢复 Matrix 会话时回退到房间默认工作目录的问题。
- 将聊天会话实际使用的 workdir 与 session 状态一起持久化，确保同一 room/user 会话继续在原工作区执行。
- 补充 SQLite 兼容迁移和重启恢复回归测试，覆盖状态存储与 orchestrator 两条链路。

### 推荐验证（可选）

1. 升级：`npm install -g codeharbor@0.1.104`
2. 检查版本：`codeharbor --version`
3. 在之前已经使用过自定义 workdir 的 Matrix room/session 中重启服务，再发送一条消息。
4. 确认机器人仍在原工作区执行，而不是回退到默认目录。
